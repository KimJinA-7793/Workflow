무엇이든-물어보살 채널
  • 1리스트에서 해당 항목 선택
    필터: Message_ID + 완료됨!!
  • 2리스트 항목 업데이트
    응답 링크 박제 + 확인함 >< 상태
결과: 확인함 >< + 응답 링크 박제
⚙️ 관리자용 핵심 로직 (유지보수 참고)
🔍 필터링의 역설: "비울수록 정확하다"

리스트 항목 선택 시 필터 조건은 Message_ID와 상태 단 2가지만 사용합니다. 다른 칸을 채우면 슬랙이 100% 일치 항목을 찾다가 에러가 납니다.

📝 Message_ID 필드 유형: 반드시 [텍스트]

리스트의 링크 유형 필드는 메타데이터가 포함된 객체 형식이라 워크플로의 순수 URL과 매칭 오류가 발생합니다. 텍스트 유형으로 설정해야 필터링 성공률 100%.

🔄 상태 값 필터링 흐름

질문 검토 기록기: 보류중 ㅠㅠ
답변 완료 기록기: 검토중 ㅎㅎ
최종 확인 기록기: 완료됨!!
각 단계가 정확한 항목만 찾을 수 있도록 상태 값을 명확히 지정합니다.

🤔 작동 방식
1

채널 양식에서 질문을 제출합니다 — 제출 즉시 질문 저장소 리스트에 자동 등록됩니다

2

담당자가 문의를 검토할 때 👀 를 추가합니다 — 리스트 상태가 검토중 ㅎㅎ 으로 자동 변경됩니다

3

응답이 스레드에 전달됩니다 — 담당자가 스레드 댓글로 답변을 답니다

4

질문에 대한 답변이 완료되면 담당자가 ✅ 를 남길 것입니다 — 리스트 상태가 완료됨!! 으로 자동 변경됩니다

5

제출자가 답변 확인 후에 확인 을 남기면, 해당 메시지가 💡 질문 저장소에 영구 기록됩니다
